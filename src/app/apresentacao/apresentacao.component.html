Website para apresentação dos serviços da empresa Tudo Fácil



Objetivo do Projeto:

  • Desenvolver website para apresentação dos serviços prestados pela empresa fictícia "Tudo Fácil"


Ferramentas Utilizadas:

  • Desenvolvido através do framework de desenvolvimento web Angular, utilizando conceitos de SPA (Single Page Application).
  • O Bootstrap foi utilizado como framework de estilos.


Inovação Proposta:

  • Realizar a inserção digital da empresa com uma aplicação moderna e responsiva
